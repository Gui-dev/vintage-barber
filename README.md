## Requirements

You need to install [Node.js](https://nodejs.org)

## How to use it

```bash
# Repository clone
$ git clone https://github.com/Gui-dev/vintage-barber.git
# Install the dependencies
$ npm ci
# Prisma Client
$ npx prisma generate
# Run the web server
$ npm run dev
```

The application will be available to access in your browser on localhost
(http://localhost:3000)
